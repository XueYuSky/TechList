# Qt Charts

## 9.1 Qt Charts概述

项目中使用方法——项目配置文件中增加:

```C++
Qt += charts
```

需要使用QtCharts的类的头文件或者源程序中使用：

```C++
#include <QtCharts>
using namespace QtCharts;
```

或者使用宏定义：

```C++
#include <QtCharts>
QT_CHARTS_USE_NAMESPACE
```
