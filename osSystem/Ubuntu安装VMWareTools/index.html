    Ubuntu安装VMWareTools

    Purpose

    本文提供了在 Ubuntu 虚拟机中安装 VMware Tools 的步骤。注意：只要操作系统（ Linux 内核）发生了更新，就需要重新安装 VMware Tools。Resolution 在 Ubuntu 虚拟机中安装 VMware Tools 的流程要在 Ubuntu 虚拟机中安装 VMware Tools，可使用如下图形用户界面或命令行界面。Install VMware Tools in Ubuntu or Ubuntu Server with a graphical user interfaceInstall VMware Tools in Ubuntu Server with only a command line interface要安装 VMware Tools，您必须挂载 VMware Tools CD 镜像，解压缩文件 (VMware Tools)，然后运行安装程序。注意：如果 VMware Tools 已经安装，则用这些步骤卸载之后重新安装。当产品更新时，VMware Tools 软件包也会更新，因此需更新已安装的 VMware Tools 版本。在 Ubuntu 或具有图形用户界面的 Ubuntu 服务器上安装 VMware Tools要挂载 CD 镜像并解压，请按以下步骤操作：启动此虚拟机。使用具有管理员权限或 root 用户权限的帐户登录此虚拟机。选择： 对于Fusion: 虚拟机 > 安装 VMware Tools 对于Workstation: VM > 安装 VMware Tools 对于Player: Player > 管理 > 安装 VMware Tools

    注意：如果您运行的是轻量版的 Fusion、不带 VMware Tools 的 Workstation 版本或 VMware Player，则系统会提示您先下载 Tools，然后才能安装它们。此时请单击立即下载开始进行下载。 打开 Ubuntu 桌面上挂载的 VMware Tools CD。右键单击类似于 VMwareTools.x.x.x-xxxx.tar.gz 的文件名，单击复制，然后选择 Ubuntu 桌面 保存文件。右键单击 Ubuntu 桌面 上的类似于 VMwareTools.x.x.x-xxxx.tar.gz 的文件名，单击解压到当前文件夹。

    vmware-tools-distrib 文件夹随即便会解压到 Ubuntu 桌面。要在 Ubuntu 中安装 VMware Tools，请按以下步骤操作：打开一个终端窗口。更多信息，请参见 Opening a command or shell prompt (1003892) 。在“终端”中，运行下面的命令以导航到 vmware-tools-distrib 文件夹：

    cd Desktop/ vmware-tools-distrib 运行下面的命令来安装 VMware Tools：

    sudo ./vmware-install.pl -d

    注意：-d 假定您希望接受默认设置。如果不使用 -d，请按 Return 接受默认值或提供您自己的答案。可使用命令 " sudo chmod u+x vmware-install.pl " 使文件可执行。输入您的 Ubuntu 密码。在 VMware Tools 安装完毕后重新启动 Ubuntu 虚拟机。

    在仅有命令行界面的 Ubuntu 服务器中安装 VMware Tools

    1. 转到虚拟机 > 安装 VMware Tools（或 VM > 安装 VMware Tools）。

    注意：如果您运行的是轻量版的 Fusion、不带 VMware Tools 的 Workstation 版本或 VMware Player，则系统会提示您先下载 Tools，然后才能安装它们。此时请单击立即下载开始进行下载。

    1. 在 Ubuntu 客户机中，运行以下命令：

    2. 运行如下命令以创建一个挂载 CD-ROM 的目录：

      sudo mkdir /mnt/cdrom

      当系统提示输入密码时，请输入 Ubuntu 管理员用户密码。

      注意：出于安全考虑，所键入的密码不会显示出来。在接下来的五分钟内您无需重新输入您的密码。

    3. 运行如下命令以挂载 CD-ROM：

      sudo mount /dev/cdrom /mnt/cdrom 或 sudo mount /dev/sr0 /mnt/cdrom

    4. VMware Tools 捆绑包的文件名因您的 VMware 产品版本而异。运行如下命令可查找确切的名称：

      ls /mnt/cdrom

    5. 运行如下命令以解压 VMwwre Tools 捆绑包文件：

      tar xzvf /mnt/cdrom/VMwareTools-x.x.x-xxxx.tar.gz -C /tmp/

      注意：x.x.x-xxxx 是在上一步中查明的版本。

    6. 运行如下命令以将目录更改到 VMware Tools 分发包：

      cd /tmp/vmware-tools-distrib/

    7. 运行如下命令以安装 VMware Tools:

      sudo ./vmware-install.pl -d

      注意：-d 假定您希望接受默认设置。如果不使用 -d，请按 Return 接受各个默认值或提供您自己的答案。

    8. 安装完成后运行下面的命令重新启动虚拟机：

    sudo reboot
